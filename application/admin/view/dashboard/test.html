功能 描述 操作
问候新人 Bot 自动向新加入群的成员打招呼
阻止非管理員添加bot 当此项激活时，群中的非管理员用户添加机器人(Bot)到群内时，该Bot将被自动删除
禁止拉人进群 当此项开启时，群仅接受主动进群的成员。被拉人进群的成员将被Bot踢除。
要求新成员提供验证信息 要求每位新入群的成员补充验证信息。成员所提交的验证信息将在 成员管理 界面中所展示。
跨群去除重复成员 如果一个新入群的成员已经存在于以下的群中，那么 Bot 将立即告知这位成员并踢除这位成员。从而确保本群和以下群内的成员保持唯一。
新人禁止各种发言 当此项激活时，新进群的非管理员成员将被暂时禁言，禁言期间无法发送任何消息。
清除成员加入的消息 当此项激活时，Bot 将自动清理新成员加入群的消息。
清除成员离开的消息 当此项激活时，Bot 将自动清理成员离开群的消息。
